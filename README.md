# 📖Project Overview

This project involved developing a web application for AI consultations, designed as a comprehensive exercise to master the fundamentals of modern web development. The primary goal was to build a functional bridge between a user-friendly interface and a robust backend, focusing on the core technologies of HTML/CSS, JavaScript, and Node.js.

# 🛠️My Role

The development was structured into two distinct phases to ensure a deep understanding of both frontend and backend principles. In the first phase, I focused on crafting the visual structure and styling using HTML and CSS, while also gaining an introductory understanding of the Bootstrap framework for responsive design. The second phase shifted the focus toward the backend, where I utilized JavaScript and Node.js to build and manage RESTful services. This phase was crucial for learning how to handle data flow, server-side logic, and API architecture, resulting in a fully integrated web application.
